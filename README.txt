===============================================
  人工智能训练师三级 · 备考资料包 使用说明
===============================================

本资料包包含以下内容：


一、官方题库（无答案）
────────────────────
官方公布的900道理论题原始题目（单选300 + 判断300 + 多选300）。
仅包含题目，不含答案，可用于自测。


二、官方题库参考答案（来自网络）
────────────────────
网络整理的900道题参考答案，不保证100%正确，足够通过考试，供刷题时对照使用。


三、人工智能3级考试理论题音频
────────────────────
900道理论题的音频版本，每道题会读出题目和答案。
适合通勤、跑步、做家务时戴耳机听，利用碎片时间加深记忆。
直接用手机/电脑的音乐播放器打开mp3文件即可。


四、刷题软件.zip —— 理论题刷题软件
────────────────────
覆盖全部900道理论题（单选 + 判断 + 多选），打开即用。

【使用方法】
  1. 解压 刷题软件.zip
  2. Windows 用户：双击「启动刷题软件（Windows双击打开）.bat」
     Mac 用户：双击「启动刷题软件（Mac双击打开）.command」
     （Mac首次打开如果提示无法打开，右键 → 打开 即可）
  3. 会自动弹开浏览器，直接开始刷题

【功能说明】
  · 题型选择：判断题 / 单选题 / 多选题，随时切换
  · 刷题顺序：支持按顺序刷 和 随机乱序刷
  · 错题本：做错的题自动进入错题库，做对后自动移出
  · 进度记忆：关掉浏览器再打开，进度还在（存在浏览器本地）
  · 如需清除进度重新刷：点击页面底部的「重置进度」按钮


五、代码填空练习.zip —— 实践题代码填空软件
────────────────────
完整还原考试中20道代码填空题的原始代码和填空位置。

【使用方法】
  1. 解压 代码填空练习.zip
  2. Windows 用户：双击「启动代码填空练习（Windows双击打开）.bat」
     Mac 用户：双击「启动代码填空练习（Mac双击打开）.command」
     （Mac首次打开如果提示无法打开，右键 → 打开 即可）
  3. 会自动弹开浏览器，直接开始练习

【功能说明】
  · 左侧题目列表选择题目（1.1.1 ~ 2.1.5）
  · 在代码中的蓝色输入框里填写答案
  · 点击「提交检查」一键判分：绿色=正确，红色=错误并显示正确答案
  · 支持按顺序刷 和 随机乱序刷
  · 进度自动保存，关掉再开还在


六、官方三级素材
────────────────────
官方提供的考试素材文件，包含实践题所需的数据文件等。


===============================================
  常见问题
===============================================

Q: 打开 .bat / .command 文件后没有反应？
A: 请确保解压后再打开，不要在压缩包里直接双击。

Q: Mac 提示"无法打开，因为它来自身份不明的开发者"？
A: 右键点击该文件 → 选择「打开」→ 在弹窗中点击「打开」即可。

Q: 刷题进度想重新开始怎么办？
A: 页面底部有「重置进度」按钮，点击即可清空重来。

Q: 关掉浏览器后进度还在吗？
A: 在，进度保存在浏览器本地存储中。但如果清除浏览器数据会丢失。
